How to use this application
1. Download gs from https://www.ghostscript.com/releases/gsdnld.html, then install it.
2. Download ps2pdf.exe tool.
3. Create a config.ini file in folder where ps2pdf.exe located and write the absolute path of ps2pdf.bat into it. it is "C:\Program Files\gs\gs10.06.0\lib\ps2pdf.bat" if you setup gs tool default.
4. Open ps2pdf.exe, select a ps file, then click covert to covert ps file to pdf file.
5. You must install .Net desktop runtime to Run this application. You can download it from https://dotnet.microsoft.com/en-us/download/dotnet.
